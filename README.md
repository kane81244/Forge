# Forge Fitness App

Forge is a native frontend fitness application built to streamline workout tracking, muscle mapping, and exercise form visualization. This project was developed as part of the NASA and Hack Club Stardance Challenge.

---

## Live Demo and Repository
* **Live Deployment Link:** https://kane81244.github.io/Forge/forge_index.html
* **GitHub Repository:** https://github.com/kane81244/Forge

---

## Tech Stack and Environment
* **Frontend:** Semantic HTML5, Custom CSS3 Grid/Flexbox layouts, Plain Vanilla JavaScript (ES6+).
* **Development Environment:** Linux Mint 22.3 Zena on a Lenovo ThinkPad T430 using VS Code.
* **Version Control:** Managed locally via Git and hosted natively on GitHub.

## Project Structure

```text
├── assets/                  # Organized media repository
│   ├── Arms Day/            # Visual cards, gifs, and diagrams for arms
│   ├── Back Day/            # Pull-day asset maps
│   ├── Cardio+Core/         # Endurance and core stability guides
│   ├── Chest Day/           # Bench press and push metrics
│   ├── Leg Day/             # Lower body biomechanics
│   ├── Shoulders Day/       # Deltoid and overhead tracking

├── forge_app.js             # Core application logic and dynamic DOM rendering
├── forge_index.html         # Main web structure and user interface
